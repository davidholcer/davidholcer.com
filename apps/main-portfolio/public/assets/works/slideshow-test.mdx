## How to Use

To create a slideshow in your MDX files, use this syntax:

```markdown
![[slideshow]][[/assets/images/your-folder]]
```

The slideshow will automatically:
- Find all images in the specified folder
- Create a carousel with thumbnail navigation
- Display images in a responsive layout
